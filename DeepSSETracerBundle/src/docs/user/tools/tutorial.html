DeepSSETracer Instruction:

DeepSSETracer is a tool for detecting secondary structures using the best GEM-UNET model that had been trained from our previous work in Cryo-EM images. It is capable of detecting helices and beta sheets in images of medium resolution

Required Input:

1. MRC: MRC map needs to be resampled to 1.0 Angstrom voxel size before running the tool, click the browse button to select the resampled MRC file.
2.Output directory: Click the browse button to select the Output path.

Resampling MRC:

To resample an MRC file to 1.0 Angstrom, we can use ChimeraX command line:
1. "open mymap.mrc"
2. "volume resample #1 spacing 1.0"
3. "save mymap-1A.mrc model #2"

Run:

After providing the necessary input, hit the run button to start the procedure, the log file will be displayed later.

Output:

Output will be the predicted helix.mrc and sheet.mrc. It will be directly displayed once the prediction finished. If the result is invisible, you may set a proper level in the panel in ChimeraX. It also can be found in the output directory.

Note:

1. The procedure takes a few seconds with GPU, and a few seconds to a few minutes without GPU. The time usage depends on the size of the image and the Computer or server. For a example, a image with size: 74*59*83 takes 81 seconds without GPU, and 7 seconds with one GPU.
2. If the images is too big to run on your computer, you may try to box chopped to smaller images to run.
3. For sample input and output, click Download.
